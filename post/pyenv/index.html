pyenv 是一个用于管理多个 Python 版本的工具。它允许你在同一台机器上安装、切换和管理不同的 Python 版本，而不需要依赖系统级的 Python 安装。

GitHub主页：

pyenv/pyenv: Simple Python version management (github.com)

pyenv-win/pyenv-win: pyenv for Windows
